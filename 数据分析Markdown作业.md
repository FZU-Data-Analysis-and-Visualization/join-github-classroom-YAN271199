## 2.代码块与公式
### （1）Python代码块
```python
import pandas as pd
# 读取本地销售数据集
data = pd.read_csv("sales_data.csv")
# 计算月度销售额平均值
avg_sales = data["销售额"].mean()
print(f"季度平均销售额: {avg_sales:.2f}万元")
```
代码功能说明：读取 csv 销售表格文件，计算所有月份销售额的平均值，保留两位小数输出结果
### (2) 均值 LaTeX 数学公式
$\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$
公式说明：用来计算一组数据的平均值，描述数据集中趋势

### (3) 方差 LaTeX 数学公式
$\sigma^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2$
公式说明：衡量数据波动大小，方差越大代表样本数据差异越大

## 三、数据分析过程
### 1. Python数据计算代码块
```python
import pandas as pd
# 构建销售数据集
sale_data = {
    "月份":["1月","2月","3月"],
    "销售额":[52,48,65],
    "订单量":[2080,1920,2407]
}
df = pd.DataFrame(sale_data)
# 计算季度平均销售额
avg_sale = df["销售额"].mean()
print(f"季度平均销售额：{avg_sale:.2f} 万元")
```
### 2. 均值统计公式
月度平均销售额计算公式：

$\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$

通过计算可得本季度平均销售额为 **55.00 万元**。

## 四、分析结论
1. 3 月销售额大幅上涨，主要原因是平台春季促销活动；
2. 2 月整体销量最低，受春节物流停运影响；
3. **运营建议**：后续每月复刻 3 月营销活动，稳定提升月度营收。
